# jschwefel-workshop

A workshop of independent tools. Each project here stands on its own — they share no
code, only a roof. They live in an organization (co-owned, so they outlast any single
account) rather than under a personal profile, and a project moves out to its own home
if it ever grows an ecosystem large enough to warrant one.

Licensing varies by project — some open source, some source-available — so check each
repo's own `LICENSE`.

## Projects

| Project | What it is |
|---|---|
| [`medit`](https://github.com/jschwefel-workshop/medit) | A native macOS text editor — a Cocoa-native take on gedit: tabbed editing, a live Markdown preview, syntax highlighting, and column/block editing. |
| [`kiln`](https://github.com/jschwefel-workshop/kiln) | A GPU-accelerated post-processing pipeline for video creators: hand a master to a Linux + NVIDIA box and get back a compact upload file plus AI-generated captions, transcript, chapters, and draft metadata, auto-archived. Source-available under BSL 1.1. |
| [`esp-idf-ds3231`](https://github.com/jschwefel-workshop/esp-idf-ds3231) | An ESP-IDF (v5.0+) component in C for the DS3231 real-time clock — time-keeping, alarms, control, and alarm interrupts wrapped in simple get/set functions. |
