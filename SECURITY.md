# Security Policy

## Reporting a vulnerability

**Please do not open a public issue for a security problem.** Report it privately through
[GitHub Security Advisories](https://github.com/jschwefel-workshop) — "Report a vulnerability" on the
affected repo — or by email to **jason.schwefel78@gmail.com**.

These are independent tools; a report is scoped to the one repo it affects. You will get an
acknowledgement, and we'll work the fix with you before any public disclosure.
